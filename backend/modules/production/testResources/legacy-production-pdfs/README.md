# Legacy production-PDF fixtures

Drop reference PDFs generated by the legacy .NET system into this directory
(ideally with multiple quantities and Unicode content). For every
`{name}.pdf`, register the exact source data the legacy PDF was generated
from under the key `{name}` in `fixtureCases` inside
`test/shop/voenix/production/pdf/ProductionPdfLegacyFixtureTest.kt`.

The test compares rendered page images within a tolerance — never raw bytes.
While this directory contains no PDFs, the comparison test skips itself and
says so on standard out.
